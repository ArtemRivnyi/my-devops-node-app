О нашем приложении

Это демонстрационное веб-приложение создано, чтобы показать полный цикл разработки и развертывания с использованием принципов DevOps.

Основные технологии, используемые в этом проекте:

  • **Node.js & Express.js:** Для создания бэкенда веб-сервера.
  • **HTML/CSS:** Для фронтенда пользовательского интерфейса.
  • **Docker:** Для упаковки приложения в переносимый контейнер.
  • **GitHub Actions:** Для автоматизации CI/CD пайплайна (сборка, тестирование, доставка).
  • **Google Cloud Run:** Для бессерверного облачного развертывания контейнера.
  • **Google Cloud Logging & Monitoring:** Для отслеживания состояния и работы приложения.

Наслаждайтесь просмотром нашего DevOps-путешествия!
